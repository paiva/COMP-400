Part 2: Prometheus DB

Prometheus DB

  • Captures in-disk and in-RAM data
    • In-disk: FileDB (Json format)
    • In-RAM: Log File (Text format)
  • Follows BIDS Protocol
  • Provides an API for each AI Module
    (Read/Write data)

Part 3: Prometheus WEB

Prometheus WEB

  • URL: prometheus-web.mcgill.io
  • Written using the Django framework
  • Serves as a visualization dashboard
  • Allows to analyze log files and visualize behavior
  • Provides RESTful api

Conclusion

  • Developed a way to capture in-disk and in-RAM data
  • Structured and organized data with BIDS format
  • Developed an internal API to read/write data to different Prometheus AI modules
  • Developed a web interface to analyze and visualize data
  • Developed a basic RESTful API via web interface
